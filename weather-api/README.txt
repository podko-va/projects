# Website Catalog

This repository contains a static website with two pages: index.html and index_3day.html.

## File Structure

- index.html: Main page of the website
- index_3day.html: Page displaying weather forecast for 3 days
- styles.css: Stylesheet file for page styling
- app.js: JavaScript file for main functionality
- app_3day.js: JavaScript file for handling 3-day weather information

## Running the Website

To run the website, simply open the index.html or index_3day.html file in any browser.
